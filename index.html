Source Maps

We also learned a bit about source maps in depth in this little endeavour. A sourcemap can be in a .map file (which is downloaded by browsers developer tools) or embedded in the built js file. The sourcemap can be either a reference to the source code files or the actual source code files embedded into it. The problem with the first alternative is that the browser needs access to the source code files, not optimal. The second approach is much better. I prefer to embedd the sourcemap (with source files included) into the built file, so everything is in one place - obviously only in development mode.
